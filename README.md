# 特点

- 前后端分离
- 好看界面（应该吧）

# 使用

1. 叉（aka Fork）下来仓库
2. 修改 `config.js`，按照提示修改字符串。
3. 修改 `custom.css`，修改配色。
4. 部署 Pages 服务，或部署到服务器。

# 注意事项

你可能需要自行搭建后端服务器。
